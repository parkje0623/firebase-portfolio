    Frenzy Quiz (Quiz Platform)

    Frenzy Quiz is a real-time interactive quiz platform enabling all users to formulate and manage various question formats. All registered users may create and host a quiz and other users can engage with these quizzes by scanning QR codes or entering a quiz ID and completing the questions. The hosted quiz provides immediate feedback of the scores to both users and a host after the completion of each question and a final leaderboard after completion of the quiz.

    source code
    Language: React JS Host server: GCP Cloud Service Technologies: Node.js, PostgreSQL, Firebase Authentication
    Pattern Matrix (Memorization Game)

    Pattern Matrix is a simple memorization game utilizing the BeagleBone, Zen Cape, and 4x4 Matrix Keypad to show embedded systems programming and hardware integration. The player will use the joystick on a BeagleBone to interact and change the general settings of the game. The 4x4 Matrix Keypad is used to display a random sequence of flashing lights that the player must memorize and re-enter the sequence to win a game.

    source code
    Languages: C
    Hardware: BeagleBone, Zen Cape, and 4x4 Matrix Keypad
